# importZIM
R function to import YARA ZIM sensor files, extracts sensor ID information from the sensor name

## Installation

Install from github using 'devtools'
```{r}
devtools::install_github("MarkusLoew/importZIM")
```

Then, use 'importZIM'

```{r}
df <- importZIM(file)
```
